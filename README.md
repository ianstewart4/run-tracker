# Run Tracker

# Install

`npm install`

---

# Things to add

- Create a `.env` file in config folder and add the following as `key = value`
  - PORT = 3000 (can be any port example: 3000)
  - DB_STRING = `your database URI`

# Run

`npm start`

## Description
Track each run to monitor progress over time.
